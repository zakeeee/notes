# 69.求平方根

**题目描述**

[69. Sqrt(x)](https://leetcode.com/problems/sqrtx/)

**解答**

```java
class Solution {
    public int mySqrt(int x) {
        if (x == 0 || x == 1) {
            return x;
        }
        int a = 0;
        int b = x;
        while (true) {
            int mid = (a + b) / 2;
            if (x / mid == mid || b - a <= 1) {
                return mid;
            } else if (x / mid > mid) {
                a = mid;
            } else {
                b = mid;
            }
        }
    }
}
```

**牛顿法**

```java
public double mySqrt(double x, double eps) {
    double res = x;
    while (res * res - x > eps) {
        res = (res + x / res) / 2;
    }
    return res;
}
```
