# 238.除自身以外数组的乘积

**题目描述**

[238.除自身以外数组的乘积](https://leetcode-cn.com/problems/product-of-array-except-self/)

**解答**



```java
class Solution {
    public int[] productExceptSelf(int[] nums) {
        int N = nums.length, tmp = 1;
        int[] pro = new int[N];
        Arrays.fill(pro, 1);
        for (int i = 1; i < N; i++) {
            tmp *= nums[i-1];
            pro[i] *= tmp;
        }

        tmp = 1;
        for (int i = N - 2; i >= 0; i--) {
            tmp *= nums[i+1];
            pro[i] *= tmp;
        }
        return pro;
    }
}
```
