# 26.删除排序数组中的重复项

**题目描述**

[26.删除排序数组中的重复项](https://leetcode-cn.com/problems/remove-duplicates-from-sorted-array/)

**解答**

双指针法。

```java
class Solution {
    public int removeDuplicates(int[] nums) {
        if (nums.length == 0) return 0;
        int i = 0;
        for (int j = 1; j < nums.length; j++) {
            if (nums[j] == nums[i]) continue;
            nums[++i] = nums[j];
        }
        return i + 1;
    }
}
```
