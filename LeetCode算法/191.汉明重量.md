# 191.汉明重量

**题目描述**

[191. Number of 1 Bits](https://leetcode.com/problems/number-of-1-bits/)

**解答**

`x = x & (x-1)` 可以去除最低位置的1

```java
public class Solution {
    // you need to treat n as an unsigned value
    public int hammingWeight(int n) {
        int count = 0;
        while (n != 0) {
            n &= (n - 1);
            count++;
        }
        return count;
    }
}
```
