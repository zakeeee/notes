# 239.滑动窗口中最大的数

**题目描述**

[239. Sliding Window Maximum](https://leetcode.com/problems/sliding-window-maximum/)

**1. 优先队列法**

维护一个优先队列（大顶堆）。

```java
class Solution {
    private PriorityQueue<Integer> heap;

  public int[] maxSlidingWindow(int[] nums, int k) {
    if (k == 0) {
      return nums;
    }
    heap = new PriorityQueue<>(k, ((o1, o2) -> o2 - o1));
    int slideCount = nums.length - k + 1;
    int[] arr = new int[slideCount];

    for (int i = 0; i < k; i++) {
      heap.add(nums[i]);
    }

    for (int i = 0; i < slideCount; i++) {
      arr[i] = findMax(nums, i, k);
    }

    return arr;
  }

  private int findMax(int[] nums, int i, int k) {
    if (i == 0) {
      return heap.peek();
    }
    heap.remove(nums[i - 1]);
    heap.add(nums[i + k - 1]);
    return heap.peek();
  }
}
```

**2. 双端队列法**

维护一个双端队列。

```java
class Solution {
  public int[] maxSlidingWindow(int[] nums, int k) {
    if (nums == null || k < 1) {
      return new int[0];
    }

    Deque<Integer> deque = new LinkedList<>();
    int[] ret = new int[nums.length - k + 1];
    int ri = 0;
    int windowBegin = 0;
    int windowEnd = k - 1;

    // deque存下标
    for (int i = 0; i < nums.length; i++) {
      while (!deque.isEmpty() && deque.peekFirst() < windowBegin) {
        deque.pollFirst();
      }

      while (!deque.isEmpty() && nums[deque.peekLast()] < nums[i]) {
        deque.pollLast();
      }

      deque.addLast(i);
      if (i >= windowEnd) {
        ret[ri++] = nums[deque.peekFirst()];
        windowBegin++;
        windowEnd++;
      }
    }

    return ret;
  }
}
```
