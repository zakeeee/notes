# 7.斐波那契数列

**题目描述**

大家都知道斐波那契数列，现在要求输入一个整数n，请你输出斐波那契数列的第n项（从0开始，第0项为0）。
n<=39

**解答**

直接循环求就可以

```java
public class Solution {
    public int Fibonacci(int n) {
        if (n == 0) {
            return 0;
        }
        if (n == 1) {
            return 1;
        }
        int a = 0, b = 1, i = 2;
        while (i++ <= n) {
            int tmp = a;
            a = b;
            b += tmp;
        }
        return b;
    }
}
```
