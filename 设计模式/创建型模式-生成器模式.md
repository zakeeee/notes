# 创建型模式-生成器模式

[TOC]

## 使用场景

生成器模式封装了一个对象的构造过程，并允许按步骤构造。使用生成器模式就不需要先创建对象然后给属性依次赋值，并且允许设置必选和可选项。

## 示例

```java
public class Role {
    private String name;
    private String race;
    private int level;
    private String appearance;

    public String getName() {
        return name;
    }

    public void setName(String name) {
        this.name = name;
    }

    public String getRace() {
        return race;
    }

    public void setRace(String race) {
        this.race = race;
    }

    public int getLevel() {
        return level;
    }

    public void setLevel(int level) {
        this.level = level;
    }

    public String getAppearance() {
        return appearance;
    }

    public void setAppearance(String appearance) {
        this.appearance = appearance;
    }

    // 生成器类
    public static class Builder {
        private String name;
        private String race;
        private int level;
        private String appearance;

        public Builder(String name, String race) {
            this.name = name;
            this.race = race;
            this.level = 1;
            this.appearance = "普通";
        }

        public Builder level(int level) {
            this.level = level;
            return this;
        }

        public Builder appearance(String appearance) {
            this.appearance = appearance;
            return this;
        }

        public Role build() {
            Role role = new Role();
            role.setName(name);
            role.setRace(race);
            role.setLevel(level);
            role.setAppearance(appearance);
            return role;
        }
    }
}
```

```java
public class Main {
    public static void main(String[] args) {
        Role role = new Role.Builder("管理员1号", "管理员").level(255).appearance("帅气").build();
        Role role = new Role.Builder("啵兔", "兔族").level(1).appearance("憨憨").build();
    }
}
```
